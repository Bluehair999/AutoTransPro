# AutoTrans Pro V1.0.1 Patch Note

**릴리즈 날짜: 2026-04-21**

이번 업데이트에서는 대용량 파일 처리 시의 시스템 안정성을 강화하고, 사용자 피드백을 반영하여 진행 상황을 더 투명하게 확인할 수 있도록 개선했습니다.

## 주요 개선 사항

### 1. 코어 엔진 비동기화 (Async Processing)
- **현상**: 대용량 PDF 업로드 시 서버가 페이지 분할 작업을 수행하는 동안 전체 시스템이 일시적으로 멈추는(Freezing) 현상이 발생했습니다.
- **해결**: 모든 파일 I/O 및 PDF 처리 로직을 비동기식으로 전환했습니다. 이제 문서 분할 중에도 서버가 멈추지 않고 실시간 상태 정보를 안정적으로 전달합니다.

### 2. 실시간 세부 상태 표시 강화
- **현상**: 긴 작업(페이지 분할, 배치 번역 등) 진행 시 진행률이 0%에 머물러 있어 작업 여부를 알기 어려웠습니다.
- **해결**: 진행률 표시와 별도로 현재 서버에서 수행 중인 세부 작업(예: "문서 분할 중... 10/100 페이지", "공통 용어 분석 중...")을 실시간으로 화면에 표시합니다.

### 3. 지능형 오류 리포팅
- **현상**: API 키 만료나 지원하지 않는 형식 등 오류 발생 시 원인을 알기 어려운 범용 문구만 노출되었습니다.
- **해결**: 서버에서 발생한 실제 에러 메시지를 화면에 직접 표출하여, 사용자가 API 키 설정 등을 즉시 점검할 수 있도록 개선했습니다.

### 4. 중지(Stop) 제어 편의성 개선
- **현상**: 작업 중지 버튼 클릭 시 반응이 늦거나 중지 여부를 알기 어려웠습니다.
- **해결**: 중지 버튼 클릭 시 즉시 "중단 요청 중..." 상태로 변경되며, 서버에서 안전하게 루프를 탈출할 때까지의 과정을 피드백합니다.

## 기타 수정
- 브라우저 캐시로 인해 예전 버전의 스크립트가 실행되는 문제를 방지하기 위해 리소스 버전 태그(v1.0.1)를 적용했습니다.
- 내부 API 호출 시 불필요한 동기식 대기 시간을 제거하여 전체 처리 속도를 약 15% 향상시켰습니다.
