# AutoTrans Pro: 전용 하이브리드 AI 번역 플랫폼 명세서

이 문서는 **AutoTrans Pro** 프로젝트의 목적, 기능, 기술적 구조를 상세히 설명하여 ChatGPT 등 AI 모델이 해당 시스템을 정확히 이해하고 학습할 수 있도록 작성되었습니다.

---

## 1. 프로젝트 개요 (Overview)

*   **프로젝트명**: AutoTrans Pro (철도4부 전용 에디션)
*   **성격**: 엔지니어링 및 철도 기술 문서에 특화된 엔터프라이즈급 AI 번역 및 OCR 플랫폼.
*   **핵심 가치**: 
    1.  **정밀도**: 기술 용어의 왜곡 없는 번역 (Glossary & TM 적용).
    2.  **안정성**: 특정 API 환경(OpenAI/Gemini)의 장애나 제한(429, 404 에러)을 극복하는 하이브리드 모델 라우팅.
    3.  **사용성**: PDF 도면이나 이미지의 레이아웃을 유지하며 원문과 번역문을 대조할 수 있는 전문적인 UI/UX.

---

## 2. 주요 기능 (Key Features)

### A. 하이브리드 AI 엔진 (Hybrid AI Engine)
*   **멀티 모델 지원**: OpenAI (GPT-4o, GPT-4o-mini)와 Google Gemini (1.5 Flash, 2.0 Flash)를 선택적으로 사용 가능.
*   **자동 최적화**: 모델별 API 버전(v1, v1beta)을 자동 감지하여 엔드포인트를 라우팅함으로써 통신 오류 최소화.
*   **비용 효율성**: 무료 쿼터가 넉넉한 Gemini와 고성능의 GPT-4o를 혼합 사용하여 가성비 극대화.

### B. 스마트 시각 번역 (Smart Vision & OCR)
*   **레이아웃 보존**: PDF, JPG, PNG 파일의 글자 위치를 인식하여 원본의 시각적 흐름을 유지하며 번역.
*   **도면 번역**: 복잡한 설계 도면 내 캡션이나 표 내부 텍스트를 AI Vision으로 정확히 추출 및 번역.

### C. 기술 용어집 및 번역 메모리 (Glossary & TM)
*   **용어집(Glossary)**: '철도4부'만의 전문 용어를 등록하여 AI가 문맥에 상관없이 약속된 용어로 번역하도록 강제.
*   **번역 메모리(TM)**: 이전에 번역된 문장을 학습하여 재출현 시 0초 만에 완벽히 일치하는 결과 제공 (비용 절감 및 일관성 확보).

### D. 실시간 모니터링 대시보드
*   **토큰 및 비용 추적**: 사용된 토큰량과 예상 비용을 실시간으로 계산하여 표시.
*   **처리 시간 측정**: 번역 시작부터 완료까지의 소요 시간을 0.1초 단위로 측정하여 업무 효율 계산 가능.

### E. 전문가용 익스포트 (Export)
*   **다양한 포맷**: 번역 완료된 결과물을 HTML(대조표 형식), MS Word(.docx), Markdown, TXT 등으로 즉시 저장 가능.
*   **보고서 최적화**: 출력물이 별도의 수정 없이 보고서에 바로 활용될 수 있도록 클린한 레이아웃 제공.

---

## 3. 기술 스택 (Tech Stack)

*   **Backend**: Node.js (Express)
*   **Frontend**: Vanilla JS, CSS (Rich Aesthetics), HTML5 (Semantic)
*   **AI APIs**: OpenAI API, Google Generative AI API (Gemini)
*   **Libraries**:
    *   `pdf-lib`, `pdf-parse`: PDF 핸들링
    *   `mammoth`: Word 문서 변환
    *   `sharp`: 이미지 전처리 및 OCR 최적화
    *   `multer`: 대용량 파일 업로드 관리

---

## 4. 프로젝트의 목표 (Project Goals)

1.  **기술 장벽 제거**: 복잡한 영문/외국어 기술 문서를 철도 전문가들이 즉시 자국어로 이해할 수 있게 함.
2.  **데이터 주권**: 외부 유출 위험을 최소화하며 부서 내부 서버(Local)에서 안전하게 작업 수행.
3.  **지식 자산화**: 단순 번역을 넘어 부서만의 용어집(Glossary)과 지식베이스를 축적하여 번역 품질을 지속적으로 향상.

---

## 5. ChatGPT 학습용 가이드라인

ChatGPT가 이 코드를 수정하거나 기능을 추가할 때는 다음 원칙을 준수해야 합니다.
1.  **Aesthetics First**: 모든 UI 변경은 프리미엄하고 모던한 디자인(다크모드, 글래스모피즘 등)을 유지해야 함.
2.  **Safety**: API 키가 노출되지 않도록 환경 변수(.env) 관리에 유의.
3.  **Robustness**: API 오류 발생 시 사용자에게 명확한 가이드를 제공하는 에러 핸들링 로직 포함.
